  2. Condensador Electrolítico 1 uF
  3. Condensador Electrolítico 47 uF
  4. Condensador Cerámico 104
  5. Diodo
  6. 3 Resistencias de 1k
  7. 2 LEDS
  8. 2 Caimanes
  9. Batería de 9v
  10. Parlante
